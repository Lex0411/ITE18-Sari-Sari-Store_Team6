# Tres Sari-Sari Store Inventory App
A desktop-based inventory system built with **Electron.js**, **HTML**, **CSS**, and **JavaScript**.

### Features
- Add, edit, and delete products
- Track inventory
- File-based local storage (JSON)
- Cross-platform desktop app
